kquant API document (0.3.6)

daily_lend_stock

주식 종목의 일자별 대차잔고 정보를 반환하는 함수

  • API URL
    • /stock/m001/loan_hist_info
    • /stock/m003/loan_hist_info
daily_lend_stock(
    symbol: 'str',
    start_date: 'DATE_IN'=None,
    end_date: 'DATE_IN'=None,
) -> pd.DataFrame

  • symbol (str): 주식 종목의 단축 종목코드

  • start_date (DATE_IN): 조회 시작일 문자열. 생략하면 조회 종료일로부터 1년 전 날짜.

  • end_date (DATE_IN): 조회 종료일 문자열. 생략하면 함수를 호출한 시점의 날짜.

  • 반환값 (pd.DataFrame): 주식 종목의 대차잔고 정보. 컬럼은 아래의 표 참조

    필드명 의미 타입
    DATE 날짜 datetime
    SYMBOL 단축코드 str
    LEND_VOLUME_NEW 대차 금일 신규 거래량 int64
    LEND_VOLUME_REPAY 대차 금일 상환 거래량 int64
    LEND_VOLUME_CHG 대차 잔고량 전일대비 int64
    LEND_VOLUME_PREV 전일 대차 잔고량 int64
    LEND_VOLUME_ACC 대차 잔고량 int64
    LEND_RATIO 대차잔고비율 int64
    LEND_AMOUNT_ACC 대차 잔고금액 int64
    LEND_AMOUNT_CHG 대차 잔고금액 전일대비 int64

import kquant as kq
kq.daily_lend_stock("005930", "2023-01-01", "2023-01-31")
