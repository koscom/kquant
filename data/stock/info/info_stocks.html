kquant API document (0.3.6)

info_stocks

복수 주식종목들의 모든 현재 상태 정보를 출력하는 함수

  • API URL
    • /stock/m001/basic_info_all
    • /stock/m003/basic_info_all

주식종목과 ETP 상품의 코드를 같이 입력하는 경우에는 출력 데이터프레임에서 공통에 아닌 필드에는 NA에 해당하는 값이 들어갑니다.

info_stocks(
    symbols: 'list[str]',
) -> pd.DataFrame
  • symbols (list[str]): 복수 주식종목들의 단축종목코드 문자열 리스트
  • 반환값 (pd.DataFrame): 주식 종목들의 모든 현재 상태 정보를 담은 Pandas DataFrame
예제 코드
import kquant as kq
kq.info_stocks(["005930","069500"])
