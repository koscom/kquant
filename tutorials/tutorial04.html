4.3 kquant 패키지의 기술적 분석 기능

kquant 패키지는 일부 기술적 분석 기능을 제공합니다. 예를 들어 단순이동평균(Simple Moving Average)는 sma 함수로 계산할 수 있습니다.

sma 함수는 daily_stock 함수로 받은 주가 정보 데이터프레임과 이동평균 구간길이를 인수로 받고 인수로 받은 데이터프레임의 복제본에 SMA_구간길이인수 형식의 열을 추가합니다.

예를 들어 20일 이동평균을 계산하면 SMA_20이라는 이름의 열이 추가됩니다. 이동평균계산에 필요한 데이터가 부족한 구간은 NaN 값이 들어갑니다.

df_sma20 = kq.sma(df, 20)

4.4 외부 패키지를 사용한 기술적 분석

kquant 패키지는 아직 개발중인 관계로 모든 종류의 기술적 분석 기능을 제공하지 않습니다. 추가적인 기술적 분석 기능이 필요한 경우 pandas_ta, talib 등의 외부 패키지를 사용할 수 있습니다.

df["MM_20"] = df.VOLUME.rolling(window=20).median()

4.5 차트 오버레이

chart_line 및 chart_candle 함수는 overlay 인수를 통해 기본 차트에 다른 정보를 라인 혹은 마커 형태로 덧붙여 그리는 오버레이(overlay) 기능을 제공합니다.

오버레이 인수로는 그려야할 열의 이름을 문자열 형식 키(key)로, 그릴 때 사용할 세부 옵션을 딕셔너리 형식 값(value)으로 줍니다. 만약 세부 옵션을 디폴트로 주는 경우에는 값으로 빈 딕셔너리를 넣습니다.

다음 예제 코드는 20일 이동평균 정보인 SMA_20 열을 추가하는 예제입니다.

kq.chart_candle(
    df_sma20,
    overlay={
        "SMA_20": {}
    }
)

오버레이를 그리는 세부옵션은 다음과 같이 줄 수 있습니다.

kq.chart_candle(
    df_sma20,
    overlay={
        "SMA_20": {
            "line_color": "red",
            "line_width": 3,
            "line_dash": "dot",
        }
    }
)

주가 차트가 아닌 거래량 차트 부분에 오버레이를 주고 싶은 경우에는 “row” 값을 2로 줍니다.

kq.chart_candle(
    df,
    overlay={
        "MM_20": {
            "row": 2,
            "line_color": "#FFFF00",
        }
    }
)

만약 거래량 차트 아래에 추가적인 차트를 넣고 싶을 때는 row 인수의 값을 3 또는 4로 줍니다.

kq.chart_candle(
    df,
    overlay={
        "OBV": {
            "row": 3,
        }
    }
)

오버레이 세부 옵션에 대한 보다 자세한 설명은 API 문서를 참조하시기 바랍니다.

print(kq.chart_line.__doc__)

        - mode: 오버레이를 그리는 방식 문자열. 다음 문자열 사용. 생략하면 "lines"
            - "lines": 선
            - "markers": 마커
            - "lines+markers": 선 + 마커
        - line_color: 선 색상 문자열. css 색상 문자열. 생략하면 자동
        - line_width: 선 굵기 정수. 생략하면 1
        - line_dash: 선 스타일 문자열. 생략하면 "solid"
            - "solid"
            - "dash"
            - "longdash"
            - "dot"
            - "dashdot"
        - marker_color: 마커 색상 문자열. css 색상 문자열. 생략하면 자동
        - marker_size: 마커 크기 정수. 생략하면 1
        - marker_symbol: 마커 기호 문자열. 생략하면 자동
            - "circle"
            - "square"
            - "diamond"
            - "cross"
            - "x"
            - "triangle-up"
            - "triangle-down"
            - "triangle-left"
            - "triangle-right"
            - "triangle-ne"
            - "triangle-se"
            - "triangle-nw"
            - "triangle-sw"
            - "pentagon"
            - "hexagon"
            - "hexagon2"
            - "octagon"
            - "star"
    :param Optional[Union[str, pd.Series]] date:

    :param DATE_IN end_date: 종료일 정보. 생략하면 데이터프레임의 마지막 날짜
