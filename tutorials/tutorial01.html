튜토리얼 1 - kquant 패키지 설치

  1. kquant 패키지는 현재 공식 pypi 서버에서 제공하지 않으며 github 사이트 release 페이지(https://github.com/koscom/kquant/releases)에서 직접 다운로드 받아 설치하여야 합니다.

  2. github 사이트 release 페이지(https://github.com/koscom/kquant/releases)에서는 kquant 패키지 파일을 .whl 확장지를 가진 wheel 파일 형식으로 제공합니다.

  3. wheel 파일 이름은 다음과 같은 형식을 따릅니다.

    kquant-[kquant 버전 번호]-[파이썬 버전 번호]-[파이썬 버전 번호]-[운영체제 문자열].whl
    • 파이썬 버전 번호에는 cp39, cp310, cp311, 세 가지가 있으며 각각 파이썬 3.9, 3.10, 3.11 버전용입니다.
    • 운영체제 문자열에는 linux_x86_64, macosx_11_0_x86_64, win_amd64, 세 가지가 있으며 각각 리눅스, 맥(Mac OSX), 윈도우(Windows)용입니다.

  4. 파이썬을 사용할 수 있는 터미널(콘솔) 창을 띄운 뒤 다음 명령으로 현재 자신이 사용하는 파이썬 버전을 확인합니다. 자신이 사용하는 파이썬 버전을 이미 알고 있는 경우에는 이 과정을 생략할 수 있습니다.

    $ python -V                 
Python 3.10.10

  5. github 사이트 release 페이지(https://github.com/koscom/kquant/releases)에서 자신이 사용하는 운영체제와 파이썬 버전에 맞는 wheel 파일을 찾아 다운로드 합니다.

  6. 터미널(콘솔) 창에서 cd 명령으로 방금 다운로드한 wheel 파일이 저장된 디렉토리로 이동합니다.

    $ cd \다운로드디렉토리

  7. pip install '자신에게해당하는wheel파일.whl' 명령으로 해당하는 .whl 파일을 설치합니다. 예를 들어 파이썬 3.10용 윈도우즈 환경에 kquant 버전 0.2.5를 설치하는 경우에는 다음과 같은 명령을 실행합니다.

    $ pip install kquant-0.2.5-cp310-cp310-win_amd64.whl

  8. 정상적으로 설치가 된 경우에는 다음과 같은 메세지가 출력됩니다.

    $ pip install kquant-0.2.5-cp310-cp310-win_amd64.whl
Processing ./kquant-0.2.5-cp310-cp310-win_amd64.whl
(... 중간 생략 ...)
Installing collected packages: kquant
Successfully installed kquant-0.2.5
$

    설치가 정상적으로 되지 않는 경우에는 현재 운영체제 및 설치된 파이썬 버전이 설치하려는 .whl 파일 이름과 일치하는지 확인해 주세요.

  9. 터미널(콘솔) 창에서 파이썬을 실행하여 다음 명령으로 kquant 패키지가 정상적으로 임포트되는지 확인합니다.

    $ python
Python 3.10.10 | packaged by conda-forge | (main, Mar 24 2023, 20:17:34) [Clang 14.0.6 ] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> import kquant
>>> kquant.__version__
'0.2.5'
>>> exit()
$
